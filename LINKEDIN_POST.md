# LinkedIn Post — TechX Website

---

🚀 Excited to share my latest project — **TechX Pvt Ltd's official website**, built from the ground up!

Over the past few weeks, I designed and developed a full-stack corporate website for **TechX Pvt Ltd** — a technology company specializing in solar energy, IT infrastructure, and power solutions.

Here's what I built:

🌐 **Multi-page Next.js web application** with dynamic routing
📬 **Fully functional Contact Form** — submissions are emailed directly to the team and auto-replied to the client
💾 **Database integration** using Neon PostgreSQL + Prisma ORM to store every inquiry
📱 **Responsive design** that works seamlessly on mobile, tablet, and desktop
⚡ **Smooth animations** using Framer Motion for a premium user experience
🚀 **Deployed on Vercel** with CI/CD via GitHub — every push auto-deploys

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 16 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS |
| Animations | Framer Motion |
| Email Service | Resend API |
| Database | Neon PostgreSQL |
| ORM | Prisma |
| Deployment | Vercel |
| Version Control | Git + GitHub |
